## link deployer dsc bot
basically makes an embed in a set channel which can be changed with a command. users can click the button linked to the embed and get a link for the site. the links are randomized. users can only get one of each link, and if they have alr gotten every link already, then they wont be able to get more. 

### commands
 - !listlinks - list all links
 - !resetuser - resets a specific users data
 - !setchannel - sets the channel where the embed will be made
 - !removelink - removes a link. links are listed out by number based on time added.
 - !addlink - adds a link.

all of these commands can only be run by the set owner in the bot-config.json file.

### prerequisites
- bun
- discord bot
